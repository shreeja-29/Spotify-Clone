# Spotify-Clone
🎵 Spotify Clone – Static Music Player
This is a Spotify-inspired static web project that mimics the core UI and basic playback interface of Spotify. It includes artist-specific pages and basic interactivity using HTML, CSS, and JavaScript.

🌟 Features
🎧 Multiple music pages: Hinana, Kalki, Shyam Singha Roy

🖼️ Clean and visually engaging UI

🎵 Local music play functionality using JavaScript

🧭 Navigation between different artist pages

📱 Responsive design with CSS

🛠️ Tech Stack
HTML5

CSS3

Vanilla JavaScript
